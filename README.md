# crypto-dashboard
Cryptocurrency Time Series Analysis Dashboard using Streamlit

## Project Description
This project analyzes cryptocurrency price data using time series techniques and visualizes trends using a Streamlit dashboard.

## Features
- Price trend visualization
- Moving averages
- Volatility analysis
- Multi-crypto comparison

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly

## How to Run

pip install -r requirements.txt
streamlit run app.py
